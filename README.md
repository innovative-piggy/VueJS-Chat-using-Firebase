# VueJS Chat using Firebase

## Project Overview
```
This is the Text Chat webapp based on VueJS & Firebase(Cloud Filestore for DB).
```

### Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
